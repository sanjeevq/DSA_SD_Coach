# Aaroh Student Setup — Start Here

1.
Create a new ChatGPT Project called Aaroh Interview Coach. Projects are used because they can keep custom instructions, uploaded files, and chats together as shared context.

2.
Open 01_CHATGPT_PROJECT_INSTRUCTIONS.md.
Fill the Student Profile section once:
   Target role:
   Target company:
   Preferred programming language:
   Experience level:
   Weak areas:
   Interview timeline:

3.
Copy the full contents of 01_CHATGPT_PROJECT_INSTRUCTIONS.md.
Paste it into the Project’s Instructions field.

4.
Upload 02_AAROH_KNOWLEDGE_PACK.md to the Project as a file/reference document. 
Do not paste this file into the instructions.

5.
Start one new chat inside the same Project.
Title or rename that chat Aaroh Progress Log.
Open 03_AAROH_PROGRESS_LOG.md.
Paste its starter content into the Aaroh Progress Log chat once.
Do not create a new progress-log chat after every practice session. Keep using the same Aaroh Progress Log chat.

6.
To start practicing, use prompts like:
   Let's practice a medium graph problem.
   Mock interview me for Google.
   Help me solve LeetCode 15.
   System design practice: design Twitter.
   Review my solution.
During DSA practice, use the hint system honestly:
   hint = small intuition only
   hint hint = stronger direction
   hint hint hint = pattern reveal
Aaroh is designed to coach Socratically instead of giving full answers immediately

7.
After every DSA, system design, mock interview, or code review session, ask in that practice chat: Give me my Aaroh Progress Update.
Copy the progress update Aaroh generates.
Paste that update into the same existing Aaroh Progress Log chat as a new message.
After 5 practice sessions, you should have:
one Project: Aaroh Interview Coach
one ongoing progress chat: Aaroh Progress Log
five progress updates inside that same progress-log chat
separate practice chats for the actual sessions


8.
To review old problems, ask from any chat inside the Project: Run my SRS review.

9.
To decide what to practice next, ask: What should I practice today based on my progress log?


Keep all Aaroh work inside the Aaroh Interview Coach Project so the coach can use the profile, knowledge file, practice history, and progress log together
