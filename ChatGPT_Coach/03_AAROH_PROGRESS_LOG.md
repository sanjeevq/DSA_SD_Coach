# Aaroh Progress Log

This file answers: **Where is the progress log?**

The progress log is one durable place named **Aaroh Progress Log**. It is where students preserve Aaroh-generated progress updates, pattern cards, design cards, mock debriefs, and SRS dates.

## Recommended setup

Use one of these options:

1. **Dedicated Project chat** — create a new chat inside the Aaroh Project, paste the starter message below, and keep appending progress updates there.
2. **Canvas** — create a Canvas named **Aaroh Progress Log**, paste this file into it, and append updates there.
3. **Markdown file** — keep this file locally, append generated updates below, and upload the latest version when asking for SRS/recommendations.

For maximum reliability, do reviews and recommendations inside the Progress Log chat or Canvas. If a separate practice chat cannot see the log, paste the latest relevant entries.

## Starter message for dedicated Project chat

```text
This chat is my Aaroh Progress Log.

Aaroh, treat this chat as the source of truth for my interview-prep progress: solved DSA problems, system design sessions, mock interview debriefs, code review notes, scores, pattern cards, design cards, weak areas, and SRS review dates.

When I paste an Aaroh Progress Update here, treat it as saved progress. When I ask for SRS review or recommendations, use these entries first. If you cannot see prior entries from another chat, say that directly and ask me to paste the relevant recent log entries. Do not invent progress.
```
